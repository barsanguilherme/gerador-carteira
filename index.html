Regras de negócio

  • Padrão: 5 dígitos CC + 2 dígitos de sequência. Ex: CC 15208 → 1520801, 1520802...
  • Código 00 é reservado para o supervisor do polo.
  • Códigos de carteiras inativas nunca são reutilizados — o gerador sempre usa max(seq)+1.
  • Após criar a carteira na hierarquia, atualize o arquivo Excel e gere um novo HTML.
  • CCs com sequência 99 estão no limite — consulte a gestão antes de criar nova carteira.
